HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • There are multiple heading tags which are h1, h2, h3, h4, h5, & h6
  • Each HTML element needs to have an opening and closing tag.
  • In an HTML file the meta element needs to be present because it allows the page to contain the information about data on the page.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Cascadaing Style Sheets (CSS) allows for the individual to make a website more readable and easier for the user
  • CSS allows the user to style a webpage 3 different ways: Inline, External & Internal Style Sheets.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • In order to make sure your local branch is up to date use the git command; git pull origin.
  • to push changes to the remote repository use the git command; git push origin.
  • A git repository allows a user or a team to collaborate on projects much easier
  • In order to create a new git feature branch the git command to use is; git checkout -b.

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • JavaScript allows the users interativity to be easier and smoother.
  • A function is a set of instructions that informs the computer on how to perform certain tasks.
